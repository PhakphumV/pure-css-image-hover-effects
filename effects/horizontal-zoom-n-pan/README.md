# Horizontal Zoom-n-pan

A CSS hover effect that scales the image upward while shifting it slightly to the side, creating a subtle zoom-and-pan motion.

## Inspiration

This variation is adapted from the horizontal zoom-n-pan example in the W3Bits CSS image hover zoom collection, with credit to Rahul at https://github.com/rahuldotdev/.

## Browser Support

| Feature | Support |
|---------|---------|
| CSS `transform` | Chrome 36+, Firefox 16+, Safari 9+, Edge 12+ |
| CSS `transition` | Chrome 36+, Firefox 16+, Safari 9+, Edge 12+ |
