Radial In

Image reveals from center inward on hover using circular radial mask transition.

Preview

Hover over the image to see the effect

Credits

PhakphumV Creator Radial reveal effect

HTML

<div class="radial-in">
  <img src="your-image.jpg" alt="Description">
</div>

CSS

.radial-in {
  overflow: hidden;
  display: inline-block;
  border-radius: 8px;
  position: relative;
}
.radial-in img {
  display: block;
  width: 100%;
  height: auto;
}
.radial-in::before {
  content: '';
  position: absolute;
  top: 50%; left: 50%;
  width: 100%; height: 100%;
  background: white;
  border-radius: 50%;
  transform: translate(-50%, -50%) scale(1.5);
  transform-origin: center;
  transition: transform 0.6s ease;
  pointer-events: none;
  z-index: 1;
}
.radial-in:hover::before {
  transform: translate(-50%, -50%) scale(0);
}
