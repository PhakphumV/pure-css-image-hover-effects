Slide Reveal

Image slides to the left while scaling up on hover.

Preview

Hover over the image to see the effect

Credits

PhakphumV Creator Initial implementation

HTML

<div class="slide-reveal">
  <img src="your-image.jpg" alt="Description">
</div>

CSS

.slide-reveal {
  overflow: hidden;
  display: inline-block;
  border-radius: 8px;
}
.slide-reveal img {
  display: block;
  width: 100%;
  height: auto;
  transition: transform 0.4s ease-out;
}
.slide-reveal:hover img {
  transform: translateX(-15%) scale(1.15);
}
