Vertical Zoom-n-pan

A smooth zoom-and-pan effect that pushes the image gently downward while scaling up.

Preview

Hover over the image to see the effect

Credits

Rahul Inspired by Adapted from the W3Bits vertical zoom-n-pan concept and credited to rahuldotdev.

HTML

<div class="vertical-zoom-n-pan">
  <img src="your-image.jpg" alt="Description">
</div>

CSS

.vertical-zoom-n-pan {
  overflow: hidden;
  display: inline-block;
  border-radius: 12px;
}

.vertical-zoom-n-pan img {
  display: block;
  width: 100%;
  height: auto;
  transition: transform 0.55s ease;
}

.vertical-zoom-n-pan:hover img {
  transform: scale(1.18) translateY(6%);
}
