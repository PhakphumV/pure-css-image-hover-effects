Slide Reveal Right

Image slides to the right on hover.

Preview

Hover over the image to see the effect

HTML

<div class="slide-reveal">
  <img src="your-image.jpg" alt="Description">
</div>

CSS

/* placeholder */
.slide-reveal-right {
  overflow: hidden;
  display: inline-block;
  border-radius: 8px;
  text-align: right;
}
.slide-reveal-right img {
  display: block;
  width: 0;
  height: 100%;
  transition: width 0.4s ease-out;
}
.slide-reveal-right:hover img {
  width: 100%;
}
